ABOUT THIS DEMO

Variant → Website → TikTok Pipeline

This demonstrates the complete workflow from Variant.com design exports through website implementation to TikTok video content.

Three Website Approaches:

Strict Single-Page: Pixel-perfect Variant copy — ALL styles exactly from export, only text content changed. No simplification.
Simplified Multi-Page: Functional marketplace with navigation, multiple pages, readable content. Design simplified for usability.
Strict Multi-Page: Best of both — full marketplace functionality with exact Variant styling preserved.

Complete Pipeline Includes:

  • Variant JSON exports with 3 design variations
  • 3 website implementations (strict, simplified, combined)
  • 5 TikTok video variants (4×15s, 1×30s, 9:16 format)
  • Remotion compositions with strict Variant styling
  • Ready-to-upload MP4 files for social media
