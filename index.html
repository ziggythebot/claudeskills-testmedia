ABOUT THIS DEMO

Variant → Website → TikTok Pipeline

This demonstrates the complete workflow from Variant.com design exports through website implementation to TikTok video content.

Two Approaches:

Strict Preservation: Follows variant-workflow rules — copy ALL styles exactly from Variant export, only change text content. No simplification, rounding, or cleanup.
Multi-Page Site: Builds a functional marketplace with navigation, multiple pages, and readable content. Design is simplified for usability.

Complete Pipeline Includes:

  • Variant JSON exports with 3 design variations
  • Next.js components with A/B testing
  • TikTok scripts (Hook/Problem/Solution/Proof/CTA)
  • Remotion video compositions (30s, 9:16 format)
